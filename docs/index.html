European rural tourism efficiency

A reproducible pipeline that combines Data Envelopment Analysis, censored regression and explainable machine learning.

Abstract Data DEA Tobit Machine learning Explainability Reproducibility

Abstract

This project measures the technical efficiency of the rural tourism industry in 33 European countries and identifies the structural drivers of cross-country differences.

Sample

33 European countries, year 2025.

Models

4 DEA + 2 censored regressions + 6 ML learners + stacked ensemble.

Code

Python · scikit-learn · XGBoost · LightGBM · SHAP · scipy · statsmodels.

Data

The panel combines the original DEA inputs/outputs, the four Tobit regressors and eleven external structural indicators.

Block Variables Source
DEA inputs Beds, establishments, employees Eurostat
DEA outputs Travellers, overnight stays Eurostat
Tobit regressors Seasonality, length of stay, protected hectares, tourist pressure Authors' compilation
Structural extension 15 structural indicators World Bank · UNESCO · CIA Factbook

DEA results

Each country is treated as a DMU; CCR, BCC, super-efficiency and SBM are solved as linear programs.

Censored regression of efficiency

A right-censored Type-I Tobit explains CCR scores from four structural regressors.

yi* = xi'β + εi,    εi ~ 𝒩(0, σ2),    yi = min(yi*, 1)

Tobit estimates and bootstrap-corrected coefficients are persisted as CSVs under results/tables/.

Machine learning

Six learners are trained on the CCR target and evaluated under leave-one-out cross-validation.

  • Elastic Net · linear baseline with L1+L2 penalty
  • Random Forest · bagging of decision trees, robust to noise
  • Gradient Boosting · sequential trees
  • XGBoost · regularised gradient boosting
  • LightGBM · histogram-based boosting
  • Neural Network · MLP (64-32, ReLU)
  • Stacked ensemble · Ridge meta-learner

Reproducibility

The full pipeline runs from a clean clone in five minutes: 

pip install -r requirements.txt

python scripts/01_prepare_data.py        # tidy CSVs
python scripts/02_run_dea.py             # DEA scores
python scripts/03_run_tobit.py           # Tobit + bootstrap
python scripts/04_fetch_external_data.py # World Bank features
python scripts/05_run_ml_models.py       # ML + SHAP + ensembles
python scripts/06_make_figures.py        # bilingual plots

All artefacts are written under results/; figures are produced in both English and Spanish.
